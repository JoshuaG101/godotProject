# Quakelike Controller
A first-person controller for Godot 4.3 with mechanics inspired by Titanfall 2 and Quake.

# Disclaimer
These scripts have been released directly from a personal project. The code is not up to a professinal standard (no docstrings, minimal comments, commented-out code). I'm uploading this so people can play around with it, just keep in mind this isn't modular or polished internally at the moment.

This controller used Garbaj's [godot_updated_controller](https://github.com/GarbajYT/godot_updated_fps_controller/tree/main) as a base.

# Installation:
Drag and drop the Quakelike Controller folder into your project

Install the [Shaker](https://godotengine.org/asset-library/asset/3137) plugin by Eneskp3441

Add the following actions to your project's input map:
- forward, left, right, backward, jump, shoot, crouch
